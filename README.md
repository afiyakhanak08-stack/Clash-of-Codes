# Clash-of-Codes
This C program is a simple Rock-Paper-Scissors game where you play against the computer in a best-of-three format. You choose r (rock), p (paper), or s (scissors), while the computer picks randomly. After each round, it shows the result and, at the end, displays scores and the overall winner.
